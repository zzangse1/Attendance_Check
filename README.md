# 출결 체크 시스템

## 소개
본 프로젝트는 개인이 다수를 직접 출결을 수동으로 체크할 수 있는 시스템입니다. 
Java 언어를 기반으로 개발되었으며, 데이터베이스는 MariaDB를 사용합니다.

## 기술 스택
- **언어**: Java
- **데이터베이스**: MariaDB (개인 서버 사용)
- **프레임워크 및 라이브러리**: 필요에 따라 추가
- **빌드 툴**: Maven 또는 Gradle (선택 가능)

## 주요 기능
- 사용자 로그인 및 인증
- 출결 체크 기능 (수동 입력)
- 출결 기록 조회 및 관리
- 그룹 생성가능
